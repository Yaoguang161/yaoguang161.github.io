构造函数

内存分配

  • 栈区: int x = 0; int * p = NULL;
  • 堆区: int *p = new int[20];
  • 全局区: 存储全局变量及静态变量
  • 常量区: string str = “hello”;
  • 代码区: 存储逻辑代码的二进制

构造函数的规则和特点

  • 构造函数在对象实例化时被自动调用
  • 构造函数与类同名
  • 构造函数没有返回值
  • 构造函数可以又多个重载形式
  • 实例化队形时仅用到一个构造函数
  • 当用户没有定义构造函数时,编译器自动生成一个构造函数

初始化列表

初始化列表特征
  • 初始化列表优先于构造函数执行
  • 初始化列表只能用于构造函数
  • 初始化列表可以同时初始化多个数据类型

拷贝构造函数

  • 定义格式: 类名 (const 类名 & 变量名)

析构函数

  • 如果没有自定义析构函数则系统自动生成
  • 析构函数在对象销毁时自动调用
  • 析构函数没有返回值,没有参数也不能重载
